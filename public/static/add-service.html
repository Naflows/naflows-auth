NASS Services

Connecting to the NASS API requires adding a NASS Service. While this form is publicly accessible, only NASS Super Admins are authorized to add services. If you are not a NASS Super Admin, your location will be blacklisted from using NASS services. Only a NASS Super Admin can manually whitelist it again.

Please note that adding a NASS Service does not bypass the need for valid credentials and tokens when making API requests.
